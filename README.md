# Semantic Catalog Search AI
מערכת חיפוש חכמה מבוססת AI עבור קטלוג כתבי יד וחסידות.
המערכת משתמשת בחיפוש סמנטי (NLP) כדי למצוא פריטים רלוונטיים גם אם מילת החיפוש המדויקת לא מופיעה בתיאור.

## איך להריץ מקומית?
1. התקן דרישות: `pip install -r requirements.txt`
2. הרץ את האפליקציה: `streamlit run app.py`
